# 🔥 Flame Watch Smart Mobile Rover

## 📌 Overview

Flame Watch Smart Mobile Rover is an IoT-enabled intelligent robotic vehicle designed for real-time fire detection and environmental monitoring. The rover autonomously avoids obstacles, supports Bluetooth-based manual navigation, and instantly sends mobile alerts when fire is detected, improving safety through remote monitoring.

## ✨ Features

- 🔥 Real-time fire detection
- 🚨 Automatic alarm activation
- 📱 Mobile alert notifications
- 📷 Camera-based environmental monitoring
- 🤖 Autonomous obstacle avoidance
- 🎮 Bluetooth-controlled navigation
- 🌐 IoT-enabled remote monitoring

## 🛠️ Technologies Used

- Arduino UNO
- ESP8266 Wi-Fi Module
- Python
- Embedded C
- Bluetooth Module (HC-05)
- Flame Sensor
- Ultrasonic Sensor
- Buzzer
- DC Motors
- Motor Driver (L298N)
- Camera Module

## ⚙️ Working

1. The rover continuously monitors the environment using flame and ultrasonic sensors.
2. If an obstacle is detected, it automatically changes direction.
3. When a flame is detected:
   - An alarm is activated.
   - A notification is sent to the connected mobile device.
4. Users can manually navigate the rover through Bluetooth.
5. The onboard camera provides live environmental monitoring.
